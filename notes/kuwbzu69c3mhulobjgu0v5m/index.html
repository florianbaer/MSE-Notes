Accuracy

Precision

Fraction of test documents classified as c that have been correctly classified - only look at the predicted class

Pc=TPTP+FPP_c = \frac{TP}{TP + FP}Pc​=TP+FPTP​

Recall

The fraction of test document labeld as c that have been correctly classified - look a the test labels given by the data

Rc=TPTP+FNR_c = \frac{TP}{TP + FN}Rc​=TP+FNTP​

F1F_1F1​ Score

Harmoic mean of precision and recall

Confusion Matrix

Gives an overview about the performance of your model.

Accuarcy is 2/3 - sum of diagonal from top right to bottom left divided by the overall sum.

Precision: given a column (i.e., a predicted label), element on main diagonal over sum of matrix column - geneva example: 11\frac{1}{1}11​

Recall: given a row (i.e., a test label), element on main diagonal over sum of matrix row - bern example 12\frac{1}{2}21​ Only works this when test labels on the left handed side and predicted labels on the bottom - caution - it could flip!
